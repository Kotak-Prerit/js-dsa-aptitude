# 30-Day Study Plan (JavaScript, Java, DSA, Aptitude)

Each day has 5 study blocks. Times are indicative and can be adjusted.

---


## Day 1
#### (8:00–9:30) – JavaScript (Core intro & variables)
- Read: What is JavaScript, runtime (browser, Node), brief history.
- Practice: Run JS in console and a simple HTML file.
- Learn: `var`, `let`, `const`, naming rules, scope overview.
- Practice: 10 snippets with `let/const`; observe scope.

#### (9:45–11:15) – Java (Syntax warm‑up)
- Setup: Install Java + IDE.
- Revise: Basic syntax, data types, I/O, conditionals, loops.
- Practice (5 programs): Sum of array, max of 3, calculator, factorial (loop), even/odd range.

#### (12:00–1:30) – DSA (Intro)
- Read: Why data structures matter.
- Learn: Arrays, indexing, time vs space, O(1) vs O(n).
- Practice: Compute time complexity of simple loops.

#### (2:30–4:00) – Aptitude (Percentages)
- Learn/revise formulas.
- Practice: 20–25 mixed questions.

#### (4:15–5:45) – Revision / Weak areas
- JS: Re‑write notes on variables/scope.
- Java: Re‑implement 2 problems without looking.
- Aptitude: Review 5 wrong questions.
